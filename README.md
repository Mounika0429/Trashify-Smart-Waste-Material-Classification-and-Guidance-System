# ♻️ Trashify: Smart Waste Material Classification and Guidance System

## 📌 Overview

Trashify is an AI-powered Smart Waste Material Classification and Guidance System developed using Flask and TensorFlow. The application identifies waste materials from uploaded images and provides intelligent recycling guidance, disposal methods, smart bin recommendations, multilingual voice support, nearby recycling center locations, and educational resources to promote sustainable waste management.

---

# 🚀 Features

- AI-based waste material classification
- Image upload and prediction system
- Recycling and disposal guidance
- Smart bin recommendations
- Nearby dumpyard and recycling center suggestions
- Multilingual translation support
- Voice assistance using gTTS
- Educational YouTube videos for waste disposal and reuse
- Eco-friendly waste management awareness

# ⚙️ Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/trashify.git
cd trashify
```

---

## 2️⃣ Create Virtual Environment

```bash
python -m venv venv
```

---

## 3️⃣ Activate Virtual Environment

### Windows

```bash
venv\Scripts\activate
```

### Linux / Mac

```bash
source venv/bin/activate
```

---

## 4️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

# ▶️ Run the Application

```bash
python app.py
```

Application will run at:

```bash
http://127.0.0.1:8080/
```
